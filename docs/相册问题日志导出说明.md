# 相册问题日志导出说明

请按下面 3 步操作：

1. 用 USB 连接手机，并打开开发者选项里的 USB 调试。
2. 在电脑终端执行下面命令：

```bash
adb logcat -c && adb logcat -v time OnlineSDKAlbumTrace:I '*:S' > album_trace.txt
```

3. 保持命令运行，按下面路径复现问题：进入聊天页 -> 点击相册 -> 授权/进入相册 -> 选择图片或视频 -> 返回聊天页。复现完成后按 `Ctrl + C` 停止命令，并把生成的 `album_trace.txt` 发回。

说明：

- 这份日志只抓取相册链路相关内容。
- 请务必在执行命令后再开始复现问题。
